API Methods

Available methods you can call:

  • MingoChat.boot(config) - Initialize the widget
    • metadata - Client and user information (optional)
      • client - External client identifier (e.g., 'adwel')
      • user_id - User ID from external system
      • user_name - User's display name
      • user_email - User's email address
    • auth - Authentication tokens (optional)
      • access_token - Short-lived access token
      • refresh_token - Long-lived refresh token
    • hideLauncher - Hide default launcher button (optional)
    • defaultOpen - Start with widget open (optional)
  • MingoChat.show() - Show/open the widget panel
  • MingoChat.hide() - Hide/close the widget panel
  • MingoChat.toggle() - Toggle widget visibility
  • MingoChat.isOpen() - Check if widget is open
  • MingoChat.updateConfig(config) - Update configuration
  • MingoChat.destroy() - Remove the widget
